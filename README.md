A Python-based WhatsApp message scheduler that allows users to send WhatsApp messages at a specific date and time using the Twilio WhatsApp API.
This project demonstrates:
API integration
Secure handling of credentials
Date & time scheduling logic
Command-line interaction


Features:
Schedule WhatsApp messages using date & time
24-hour time format support
Secure API key handling using environment variables
Uses Twilio WhatsApp Sandbox
Simple command-line interface
